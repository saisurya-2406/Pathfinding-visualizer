# Pathfinding Visualizer

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/saisurya-2406/pathfinding-visualizer.git
   ```
2. Navigate into the project directory:
   ```bash
   cd pathfinding-visualizer
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Features
- Visualize various pathfinding algorithms.
- Interactive grid layout for pathfinding.
- Customizable grid sizes and algorithm settings.
- Real-time performance metrics and path step details.

## Algorithm Explanations
- **Dijkstra's Algorithm**: A graph search algorithm that finds the shortest path from a starting node to all other nodes.
- **A* Search Algorithm**: An extension of Dijkstra’s algorithm that uses heuristics to improve search performance.
- **Breadth-First Search (BFS)**: An algorithm that explores all neighbor nodes at the present depth prior to moving on to nodes at the next depth level.
- **Depth-First Search (DFS)**: An algorithm that explores as far as possible along a branch before backtracking.

## How to Use
1. Open the index.html file in your browser.
2. Choose an algorithm from the dropdown menu.
3. Select a grid size and start point on the grid.
4. Press the "Visualize" button to see the pathfinding in action.
